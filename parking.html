PARKING

Application web en PHP avec le framework Laravel

Un utilisateur peut réserver une place qui leur est attribuée aléatoirement,
si aucune place n'est disponible l'utilisateurs sera placé en liste d'attente en attendant qu'une reservation se termine.

De leurs côtés, les administrateurs peuvent :

- créer/modifier/supprimer des utilisateur/places
- gérer les reservations
- éditer la liste d'attente
